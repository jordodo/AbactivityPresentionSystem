Klant registratie

Vul de onderstaande gegevens in om u te registreren als klant. Hierna krijgt u een Klantcode toegekent,
deze code is belangrijk. Houdt deze bij de hand als u uw account wilt aanmaken, deze code zal uw account
dan aan uw registratie als klant koppelen. Alle velden zijn verplicht, behalve een eventueel tussenvoegsel en
een bedrijfsnaam. Immers hebben wij particulieren als klant.
